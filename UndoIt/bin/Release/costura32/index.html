App Introduction

Undo It is a fully featured, file recovery tool for Windows.

Accidentally deleted a file? Never fear; the data is probably still on your drive and may be recoverable.

Undo It finds all of the deleted files on your hard drive, flash drive or SD card and allows you to recover them.

Features
  • Scan all of the deleted files
  • Scan all of the deleted files by name
  • Scan all of the deleted files by types
User manual:

How to use Scan all feature?

Scan all feature will help you search all deleted files. Access this feature from: Undo It Scan all :

Firstly, click on Scan all button

You can browse by one of your Disks or Folder. Check on Select All Disks box if you want to scan all of your Disks at a time.

And then click Scan button to start scanning.

Click Stop button at any time when you want to stop scanning.

finished scanning, you can type the keyword (file name) in the Search box to be able to find your file more quickly.

If you also want to show system files, tick on the box named Show system file and unknown file types next to the Search box.

After found out the file you need, please click right on that file select Recovery file or click to select that file Restore file. And then select the location to save your file.

Notice: We recommend you to save file in the Disk that different from the current Disk of the file.

How to use Scan by name feature?

Scan by name feature will help you search all deleted files by file name.

Access this feature from: Undo It Scan by name :

Click on Scan by name button and do the same as Scan all button. But there are some differences as below:

  • Type the file name before scanning. (If you cannot remember the correct name, just type in the first letter of the name for scanning similar files)

How to use Scan by types feature?

Scan by types feature will help you search all deleted files by file types.

Access this feature from: Undo It Scan by types :

Click on Scan by types button and do the same as Scan all and Scan by name button. However please note:

  • You don’t need to type in file name. Just select the Disk and select the icon that represents for the file type you want to find or click on icon to find other file types.
Some wheelie good advice for you:
  • Undo It app works best if performed as soon as possible after file deletion.
  • When you delete a file, the data is not lost - but new files being written to the hard drive may overwrite your data permanently, making recovery impossible.
